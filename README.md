🧠 Redmine RAG 시스템 개요

Redmine RAG 시스템은 사내 Redmine 이슈·위키·모델 성능 기록 데이터를 자동 수집하고
이를 Vector DB 기반 검색(Retrieval) + LLM 요약/표 생성과 결합한
AI 기반 실험/모델 조회 어시스턴트입니다.

Commit logs

Model 성능 (Accuracy, F1, Dice 등)

Issue·Journal 통합 변환

정제 데이터(JSONL) 저장 → 임베딩 단계로 전달
