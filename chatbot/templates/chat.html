1. 개요

Redmine RAG 챗봇은 MTS BIO-DT 팀의 과거 실험 데이터를 자연어로 검색할 수 있는 시스템입니다.
수백 개의 Redmine 이슈를 일일이 찾아볼 필요 없이, 질문만 하면 관련된 실험 결과를 찾아드립니다.

2. 질문 예시

질문 예시 (클릭하면 자동 입력됩니다)
성능 지표 조회
"최신 TSR 모델의 Dice Score는?"
특정 모델의 성능 지표를 확인할 때
특정 버전 검색
"BRAF 모델 v0.2.1의 실험 결과 요약해줘"
버전별 실험 결과를 비교할 때
주제별 검색
"Thyroid Mutation 관련 최신 실험 3개 보여줘"
특정 주제의 최근 실험을 찾을 때
조건 검색
"F1-score가 0.8 이상인 모델들은?"
특정 조건을 만족하는 결과를 찾을 때
개선 추적
"TMR 모델의 최근 개선 사항은?"
모델의 발전 과정을 확인할 때
환경별 검색
"PyTorch 1.13 환경에서 실험한 결과 있어?"
특정 개발 환경의 실험을 찾을 때
문제 해결
"GPU 메모리 부족 오류 해결 방법 찾아줘"
과거 문제 해결 사례를 찾을 때

3. 제한사항

답변하기 어려운 질문
• 미래 계획에 대한 질문 (예: "내일 실험 뭐하지?")
• 실험 데이터와 무관한 질문 (예: "커피 맛집 추천해줘")
• 코드 생성 요청 (예: "코드 작성해줘")
• 너무 짧거나 애매한 질문 (예: "이거", "성능")

4. 효과적인 질문 작성 방법

구체적으로 질문하세요
부적절: "모델 성능은?" → 적절: "TSR v0.2.0 모델의 Dice Score는?"
원하는 정보를 명확히 하세요
부적절: "갑상선" → 적절: "Thyroid Mutation의 최신 실험 결과는?"
키워드를 포함하세요
모델명, 버전, 성능 지표, 실험 주제 등을 포함하면 더 정확한 답변을 받을 수 있습니다.
후속 질문이 가능합니다
대화 맥락을 기억하므로 "그럼 그 모델의 하이퍼파라미터는?" 같은 후속 질문도 가능합니다.

5. 참고 사항

• 답변 하단에 표시되는 "참고 이슈" 링크를 클릭하면 Redmine에서 관련 전체 이슈를 확인 가능합니다.
• 이 챗봇은 실시간 대화가 아닌 과거 데이터 검색 도구입니다.
• 정확한 수치나 상세 내용은 반드시 원본 Redmine 이슈를 확인하시기 바랍니다.
